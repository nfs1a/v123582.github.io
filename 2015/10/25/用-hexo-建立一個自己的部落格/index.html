用 hexo 建立一個自己的部落格

October 25, 2015

什麼是 hexo？

Hexo 是一個用 node 撰寫，支援 markdown 的 static blog template。

怎麼使用？

  • 安裝
$ npm install -g hexo-cli # 利用 npm 安裝 hexo
  • 建立 hexo 專案
$ hexo init <folder> # 建立一個 hexo 專案
$ cd <folder>
$ npm install
  • 寫作
$ hexo new [layout] <title>
  • 本機端發布
$ hexo server
  • 發佈 & 部署
$ hexo generate
$ hexo deploy
# or
$ hexo generate --deploy
